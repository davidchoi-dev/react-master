# 인증

Firebase 인증(Authentication) 서비스는 애플리케이션을 이용하는 사용자를 손쉽게 인증하는 다양한 방법을 제공합니다.

# Firebase 로그인 방법 설정

Firebase 인증은 이메일/패스워드 또는 외부 공급자(예: Google, Facebook, GitHub 등) 인증 방법을 제공합니다. 사용할 인증 방법을 활성화 하려면 Firebase 프로젝트 페이지에서 Authentication > 로그인 방법(Sign-in method)을 선택합니다. 이메일/비밀번호 방법으로 로그인할 수 있도록 설정하기 위해 이메일/비밀번호 상태를 '사용 설정'으로 변경 저장합니다.

# Firebase 회원가입

Firebase 인증 서비스 사용을 위해 firebase/config.js 파일 상단에서 인증 패캐지를 호출합니다.


 

import * as firebase from 'firebase/app';
import 'firebase/auth';

import { auth } from 'firebase';

handleSignUp = useCallback(async (e) => {
  e.preventDefault();

  const {email, password} = userInput;

  } catch({message}) {
    console.error(message);
  }

# useAuthState 훅

React Firebase Hooks은 Firebase Auth의 인증 상태를 처리하는 유용한 함수를 제공합니다. 이 훅은 Firebase에서 인증 상태를 검색하고 모니터링 합니다. (firebase.auth().onAuthStateChange() 메서드 래핑, 항상 최신 상태 확인)

import { useAuthState } from 'react-firebase-hooks/auth';

/**
 * AuthUser 컴포넌트
 */
function AuthUser() {
  // useAuthState 훅을 사용해 인증 사용자 정보를 가져 옴
  const [ user, loading, error ] = useAuthState(auth);

  return (
    // JSX
  )
}

loading 인증 상태가 로드 되고 있는지 여부
error 사용자 정보를 읽으려 할 때, Firebase 인증 서비스에서 반환된 모든 firebase.auth.Error 또는 undefined

# 사용법

공식 문서의 예제를 살펴보며 useAuthState 훅의 사용법을 학습합니다.

import { useAuthState } from 'react-firebase-hooks/auth';

/**
 * CurrentUser 컴포넌트
 */
const CurrentUser = () => {
  // Firebase 인증 정보를 다루는 훅
  const [user, loading, error] = useAuthState(firebase.auth());

  // 로그인
  const login = () => {
    firebase.auth().signInWithEmailAndPassword('test@test.com', 'password');
  };

  // 로그아웃
  const logout = () => {
    firebase.auth().signOut();
  };

  // 초기화 조건 처리 렌더링
  if (loading) {
    return (
      <div>
        <p>사용자 초기화 중 ...</p>
      </div>
    );
  }

  // 오류 조건 처리, 렌더링
  if (error) {
    return (
      <div>
        <p>오류: {error}</p>
      </div>
    );
  }

  // 인증 사용자가 존재할 경우, 렌더링
  if (user) {
    return (
      <div>
        <p>현재 사용자 계정: {user.email}</p>
        <button onClick={logout}>로그아웃</button>
      </div>
    );
  }

  // 위에 나열된 조건 중 어떤 것도 참이 없는 경우 렌더링
  return <button onClick={login}>로그인</button>;
};
