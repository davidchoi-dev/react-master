# props 적용

영상 강의를 시청하며 글을 읽고 실습하면 더욱 효과적입니다.

React 컴포넌트는 요소로 사용될 때 속성을 전달 받고, props 속성으로 전달 받은 속성에 접근이 가능합니다. Styled Component의 컴포넌트 또한 props 속성을 인터폴레이션(${}) 내부에 설정된 함수((props) => {})를 통해 처리할 수 있습니다.




 




import styled from 'styled-components';

const AppButton = styled.button`
  color: ${ props => props.reject ? '#f60' : '#06f' };
`

export default AppButton

<AppButton reject>취소</AppButton> // color: #f60
