# 훅(Hooks) API

react-spring은 현재 5개의 훅(hooks)을 제공합니다.

react-spring 훅 설명
useSpring 싱글 스프링으로 데이터를 a → b로 이동합니다.
useSprings 리스트에 사용하는 멀티 스프링으로 개별 아이템의 데이터를 a → b로 이동합니다.
useTrail 싱글 데이터 세트를 가진 멀티 스프링, 한 스프링이 다른 스프링의 뒤를 따릅니다.
useTransition 마운트 또는 언마운트 될 때 트랜지션 됩니다. (예: 리스트 아이템이 추가, 제거, 갱신 될 때)
useChain 멀티 애니메이션을 모두 큐(queue, 대기열)에 체인(chain) 합니다.

# 기본 사용법

제공되는 훅 중 가장 사용이 손쉬운 것은 useString 훅 입니다. 하지만 다른 훅 또한 애니메이션에 동일한 개념이 적용됩니다. 사용법을 살펴보죠.

import { useSpring, animated } from 'react-spring';

const App = () => {
  const props = useSpring( { from: { opacity: 0 }, opacity: 1, } );

  return (
    <animated.div style={props}>페이드 인</animated.div>
  )
}

# 패키지 호출

react-spring을 사용하기 위한 1단계는 패키지를 로드하는 것입니다. useSpring 훅과 스페셜 팩토리(special factory)라 불리는 animated 유틸리티 함수를 꺼내 사용할 수 있습니다.

import { useSpring, animated } from 'react-spring';
1

animated는 성능 상의 이유로 React 외부(outside)에서 애니메이션을 적용합니다.

# 스프링 정의

스프링은 시작(from) 상태에서 종료(to) 상태의 값을 애니메이션 합니다. 업데이트는 누적되며 전달된 모든 값을 기억합니다. 임의(arbitrary)의 키워드(예: from 기본 값과 같은 예약 키워드)를 사용할 수 있습니다. 관련 키워드는 API를 통해 익힐 수 있습니다. 전달 받은 props는 정적 값이 아니고, 자체적으로 업데이트 되므로 animated 유틸리티에서만 사용 가능합니다. (일반적인 <div> 등에서 사용할 수 없음)

const props = useSpring({
  // 시작
  {
    from: { opacity: 0 }
  }
  // 종료
  opacity: 1
});

# 애니메이션 값을 뷰에 연결

컴포넌트의 뷰(view) 부분에서 스프링 정의 속성인 props를 연결하면 애니메이션 설정이 마무리됩니다. react-spring의 훅을 사용해 정의한 props는 animated 확장을 거쳐 연결되어야 정상적으로 애니메이션 됩니다.

<animated.div style={props}>페이드인</animated.div>
1

컴포넌트를 참조한 변수를 요소로 사용하는 문법을 사용할 수도 있습니다.

const AnimateDiv = animated('div');

<AnimateDiv style={props}>페이드인</AnimateDiv>
1
2
3

웹 환경에서 사용될 경우, 애니메이션은 모든 HTML 요소(예: div, span, svg 등)에 적용할 수 있습니다. React 컴포넌트, 스타일 컴포넌트(styled-components) 또는 다른 플랫폼의 요소에 애니메이션을 적용하는 방법은 다소 다릅니다. 아래 예시를 살펴보세요.

// React 컴포넌트 Donut, 애니메이션 확장
const AnimatedDonut = animated(Donut)

// React Native 컴포넌트 View, 애니메이션 확장
const AnimatedView = animated(View)

// 스타일 컴포넌트(styled-components), 애니메이션 확장
const AnimatedHeader = styled(animated.h1)`
  font-size: 3rem;
  color: #0984e3;
`

# props ≠ 스타일

앞서 살펴 본 예에서 style 속성에 props를 할당하다 보니 오해의 소지가 있을 수 있습니다만, props는 style 속성에만 설정 가능한 것이 아닙니다.

const props = useSpring({
  from: { opacity: 0 },
  opacity: 1,
})

return <animated.h1 style={props}>안녕! react-spring</animated.h1>
