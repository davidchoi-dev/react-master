# React Firebase

React Firebase는 React 앱과 Firebase를 통합하는 컴포넌트 API를 제공합니다.

  • 선언적 프로그래밍
  • 컴포넌트 기반 API
  • 빠른 성능
  • 크로스 플랫폼

# 선언적 프로그래밍

별도의 데이터 구독 관리(subscription management) 걱정 없이, 데이터 패스(path)를 설정해 사용합니다.

# 컴포넌트 기반 API

보일러플레이트(boilerplate)가 없습니다. 바로 앱 로직(logic)을 작성하세요.

# 빠른 성능

앱에서 구독(subscription)하게 되면, 자동으로 재사용(re-use) 합니다.

# 크로스 플랫폼

React Native, rn-firebase과 작동됩니다.
