# 시작하기

Ediya UI 웹 사이트 학습용 데이터를 React App으로 변경하는 미니 프로젝트를 진행합니다.

# 학습 목표

미니 프로젝트를 실습하는 목표는 다음과 같습니다.

  • 순수 웹 사이트(HTML, CSS, JavaScript, DOM API) → React 앱 사이트
  • React 선언형 프로그래핑
  • React 컴포넌트/모듈 시스템
  • React 앱 스타일 관리
  • React 앱 접근성 개선
  • 프로젝트 빌드 & 배포

# 실습 파일

실습 파일 구성은 다음과 같습니다. 실습 파일을 다운로드 받아 압축을 풀어 실습을 준비합니다.

/
├── pureWebResources/ # 순수 웹 리소스
│   ├── api/
│   ├── css/
│   ├── images/
│   ├── index.html
│   └── js/
└── reactAppResources/ # React 리소스
    ├── public/
    └── src/
